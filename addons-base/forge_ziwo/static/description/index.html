Documentation & Guide

Overview

Agent screen is the central screen for operating calls, as well as viewing the history of all calls made by any agent.

System Tray Icon

The ZIWO System Tray Icon is used to quickly access your dialing panel with only one click.

Agent Screen (Overview)

You can navigate to the Call History tab in order to view the overall history of call records.

The contact field is used to search for a stored record of someone using mobile, phone, email or name.

Quick views is a summarized view of all available records in each category once a contact is selected.

Agent Screen (Contact Selected)

The option to edit contact details through the Edit Contact Wizard is available directly from the agent screen.

Click to call is also available here for fast one click calls.

The quick views are updated once the contact is selected:

Agent Screen (Edit Contact Wizard)

Quick changes to the contact details can always be made from agent screen quickly.

This avoids the need to navigate away and through multiple views to search and find the contact.

Agent Screen (Edit Contact Wizard)

Notes can also be added directly from agent screen to any record available in the quick views.

Once added, the note will show up on the corresponding record:

Call History (Overview)

Contact details show an up-to-date contact name, regardless of the mobile number.

Mobile number is a fixed field that stays the same regardless of contact changes.

Call details are also displayed with a call recording player if the call was answered.

Context/model is where you can set the context of the call.

Call History (Model/Context)

Call summary is applied once the model is changed.

This can also be applied through the settings automatically based on context. (Under Settings)

Click to Call (Overview)

Click to Call creates a call directly through one click and brings up the ZIWO panel as well.

Click to Dial opens a dial wizard where you enter the number you would like to call:

Both Click to Call and Click to Dial will retain the context of your active form if it is enabled in the settings.

For example, using Click to Dial will call a different number than the one on the form, but it will keep that active form as context.

These options are also available across Subscription and Helpdesk Tickets as well:

Settings (Overview)

History on Active Form: This will capture the context of your current form view if you’re on a call and that form is registered in Active Forms. The only available forms are Contacts, Helpdesk Ticket, Lead/Opportunity and Sales Orders (Subscriptions)

Navigate (Outbound Calls): If this is enabled, whenever an outbound call is made, it will navigate you to the agent screen and show you all the information about that contact if it is registered in the system.

Override Navigate (Outbound Calls): If this is enabled, whenever a call is made using Click to Call or Click to Dial, you will not be navigated to agent screen and it will override the setting above if it is enabled.

Navigate (Inbound Calls): If this is enabled, whenever an inbound call is made, it will navigate you to the agent screen and show you all the information about that contact if it is registered in the system.

Admin Auth Token: If this is filled in, it will be used as the default authorization token to retrieve the call recordings, especially if retrieving the call recording fails initially. (This requires retrieval of the auth token from the account name/organization admin.)

Settings (Active Forms)

• Contacts (existing contacts)

• Helpdesk Ticket (create new ticket)

• Lead/Opportunity (create new lead)

• Sales Orders (Subscriptions) (create new subscription)
